
Here are four `pandas` methods that can help with some basic data exploration :

- `.head()` & `.tail()`
- `.info()`
- `.describe()`

`.head() & .tail()` methods :

- these methods display the first 5 and last 5 rows of the DataFrame (by default)
- syntax : `df.head() and df.tail()`
- to have more than 5 rows, pass in a specific number :
- `df.head(10)   # Displays the first 10 rows`

Data types and missing values with `.info()`

- used to display the information about each column in the DataFrame
- `NaN - Not a Number` : missing values or incomplete data in the dataset 
- syntax : `df.info()`

`.decribe()` method :

- use `df.describe()` to get a printout of summary statistics ( mean, min, max, standard deviation, etc ) for every numeric column.
- add `include='all'` includes strings in the `.describe()` method - `df.describe(include='all')`
