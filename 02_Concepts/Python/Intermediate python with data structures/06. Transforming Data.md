
`map()` function -

- syntax : `map(function, data)
- performs function on each element of the data
-       `def divide_by_2(x):` 
-       `return x / 2`
-       `halved_numbers = map(divide_by_2, [1, 2, 3, 4, 5])` 
-       `print(list(halved_numbers))`
-       `Output: [0.5, 1.0, 1.5, 2.0, 2.5]`
- **Note:** In order to get the actual list, we need to use the `list()` function. This also goes for the `filter()` function

`filter()` function - 

-  we select elements from the `data` that satisfy the given condition specified by the `function` and return a new list.
- example : 
     def `filter_even(x): 
        return x % 2 == 0 
     even_numbers = filter(filter_even, [1, 2, 3, 4, 5]) 
     print(list(even_numbers)) 
     Output: [2, 4]
     the function will go through the list of number and if `x % 2 ==0` the function `filter_even` is true it will print the number.

`reduce()` function - 

- from functools import reduce
- `reduce()` takes a collection of iterable `data` and combines its elements into a single value via a `function`. An optional `initial` value can be used, as well.
- `reduce()` Example:
     from functools import reduce 
     def multiply(x, y): 
        return x * y 
     product = reduce(multiply, [1, 2, 3, 4, 5]) 
     print(product)
