
Comma separated values (CSV) -
- format to organize the data
- each row represents a record
- values within a record is separated by commas

.reader ( ) method -
- used to read or display file's content, row by row
- used to create csv reader object
- try adding ('filepath', 'mode', encoding='utf-8') to reduce errors

.writer( ) method -
- method to create csv writer object
     with open('output.csv', 'w', newline=' ') as file:
     csv_writer = csv.writer(file)

.writerow/s( ) method and newline parameter -
-  method can parse something iterable (like a list of lists!) and write it into the CSV file
- newline='' ensures results are printed in a new line.
     csv_writer.writerows(data_to_write)

header in csv file -
- reader( ) is iterative, to skip the first row
- use : 
     header_variable = next(csv_reader)
     csv_file.readline( )
