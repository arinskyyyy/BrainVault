
- transforming complex loops into elegant one-liners
- **List comprehensions** can transform data, filter elements, and even create nested lists
- Syntax :
     [expression for item in dataset if condition]
1. The `expression` is applied to each `item`.
2. A `for..in` statement loops through each `item`.
3. An optional `if condition` can filter out items.

- **Note:** A list comprehension works with `for` loops but not `while` loops

- Syntax breakdown :
     `even_numbers = [num % 2 == 0 for num in numbers]
- this statement simply checks whether the elements in numbers list is even or not, if yes then return "True" or "False".
     `even_numbers = [num for num in numbers if num % 2 == 0]`
- Here, `num` holds the elements returned by `if condition` which are filtered .
- `for num in numbers` simply loops through each item/ element in the list.
- `huh = [print("Fantasy names: ", names) for names in random_names]`

lambda - 
- one line statement to define a function
- syntax : lambda arguments( function parameter ): expression
- example : 
     `double = lambda x: x ** 2`
- used in map and filter functions :
     `double = list(filter(lambda x: x ** 2, dataset))`
     `double = list(map(lambda x: x ** 2, dataset))
