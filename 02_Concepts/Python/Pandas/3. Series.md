
Access specific column using - `movies.genre or movies['genre']`

Pandas Series -

- series is like a single column of a dataframe table
- check using : `type()` - `type(movies) and type(movies['genre'])`

Accessing multiple columns -

- syntax : `variable = characters[['name', 'race']]`
- this will DataFrame rather than a Series
- notice the double brackets in the syntax, that is a list passed to the characters DataFrame, normally to access a specific series, use `variable = characters['name']`

Accessing all but one column with `.drop()` - 

- sometimes we want everything except for one column, use `.drop()`
- `variable = characters.drop("column", axis = 1)`
- here, dropping the mentioned column and `axis = 1` tells the pandas to drop the column rather than a row
- to access column (down arrow) use : `axis = 1` and for rows (right arrow) use : `axis = 0`
- to drop multiple columns use : `df.drop(columns=['B', 'C'])`
- if you want to modify the existing dataframe instead of creating a new variable use : `df.drop(columns=['B', 'C'], inplace=True)`

**Note :** if you want to ignore errors for missing columns, you can add `errors='ignore'`.
