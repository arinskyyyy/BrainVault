
We’ve learned how to select specific columns, but what if we only want certain **rows**?

how to filter for movies longer than 120 minutes :

- `long_movies = movies[movies['runtime_minutes'] > 120]`

 why is it double? movies is mentioned two times - 
 
- using : `movies['runtime_minutes' > 120]` will result in comparing and returning bool, whether it is True or False.

- to filter, will add `movies['runtime_minutes'] > 120` inside `movies[]`

- where the inside `movies part` will result in showing the bool series and the outer `movies part` will keep only the rows where the condition is `True`

**AND and OR**

- Can filter based on multiple conditions using `AND(&) and OR(|)`

- example :
-       something = movies[(movies['runtime_minutes'] > 120) & (movies['genre'] == 'Sci-Fi')]
-        something = movies[(movies['nothing'] == 'what') | (movies['something'] == 'Yes?') | (movies['everything'] == 'No?')]
