
Defining to make a particular column an index :

- syntax : `df = pd.read_csv('filename', index_col='Pokemon Name')`
- Now the column `Pokemon Name` is the index for the whole dataframe

`.loc[INDEX, COLUMNS]` method :

- gives the column details of the particular searched index
- example : `df.loc['Pikachu', 'Height']`

- for multiple columns : `df.loc['Pikachu', ['Height', 'Weight', 'Speed']]`
- use slicing :
   
  if columns are right next to each other, selects all columns between 'Type 1' and 'Speed' (inclusive) for Pikachu : 
   
  `df.loc['Pikachu', 'Type 1':'Speed']`

Maximum :
- finding max speed : `df['Pikachu'].max() or df.max() or df.desribe()`

Pokemon with speed of 200 :
- using sorting : `df.sort_values(by=[COLUMN], ascending=False)`
- simple method : `df.loc[df['Speed'] == 200]`

Common speed :
- syntax : `df['Speed'].value_counts().head()`

Shortest named pokemon with speed 200:

- syntax : `df.loc[df['Speed'] == 200].sort_values(by='Pokemon name')`
- or : `df.loc[(df['Speed'] == 200), 'Pokemon name'].min()`
