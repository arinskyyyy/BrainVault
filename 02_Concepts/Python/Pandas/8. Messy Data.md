
# Data Cleaning

Missing data :

   - before diving into cleaning, requires one to know the missing values 
   - syntax : `.info() method`

Two options to handle missing data :

Option I :
- remove rows with `.dropna()`
- this removes any rows which contain at least one missing value
- use case : rows information is not important
- if one cares about the missing values in some rows, use :
```
cleaned_df = df.dropna(subset = ['username', 'location'])
```
- This removes rows that have a missing value in the `username` or `location` columns.

Option II :
- replace missing values with `.fillna()`
- **Imputation** means filling in the blanks with a value that makes sense. Your choice on how to do this will depend on the data. Some common choices include:
  -A default value, like `'Unknown'`
  -The most common value (the mode)
  -The average value (if your data is numerical)

The following code would fill in the missing values for the `location` column with `'Unknown'`:

```py
df['location'] = df['location'].fillna('Unknown')
```

Finally, assuming the data is _numerical_, we can fill empty values with the average using `.mean()`.

```py
mean_age = df['age'].mean()

df['age'] = df['age'].fillna(mean_age)
```
