
Find duplicate data : 
- syntax : `df.duplicated()`
- to see rows instead of series of true / false : `df[df.duplicated()]`

Remove duplicates :
- syntax : `something = df.drop_duplicates()`
- Removes rows that have a duplicate ISBN with another row :
```py
unique_books = catalog.drop_duplicates(subset = ['ISBN'])
```

Inspect data types :
- syntax : `df.dtypes`
- returns, series of data types for each column 

Changing data types :
- to convert from string to int, syntax : `df['age'] = pd.to_numeric(df['age'])`
- will only work if all rows in the column have numeric value, if string, then throws an error
- to solve : `df['age'] = pd.to_numeric(df['age'], errors = 'coerce')`
- will result in converting string to NaN (not a number)
