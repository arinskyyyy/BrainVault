
There are a handful of very helpful `.str` methods for basic string processing:

- `.str.lower()`: makes everything lowercase
- `.str.upper()`: makes everything uppercase
- `.str.strip()`: removes whitespace from both ends
- `.str.contains('text')`: checks for a substring
- `.str.replace('old', 'new')`: replaces parts of the text

 Example : `df['nothing'] = df['nothing'].str.lower()`
