**CSS**, or **C**ascading **S**tyle **S**heets, is a styling language that paints a website with colors, fonts, layouts, and animations.

The programs we write will be files with the extension **.css**.

### **How to link CSS to Html?**

Can link using `link href="style.css"` in html :

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <link href="styles.css" rel="stylesheet" />
    <title>Musicodes NYC</title>
  </head>
  
  <body>
  </body>
</html>
```

In HTML, `rel` stands for **relationship**.

When you write `rel="stylesheet"`, you are telling the web browser : "The relationship between this current HTML document and the file I am linking to is that the linked file is a **stylesheet**."

> [!IMPORTANT]
> If you just used `<link href="styles.css" />` without the `rel` attribute, the browser would likely ignore the file. It needs `rel="stylesheet"` as the specific instruction to take the CSS code inside `styles.css` and use it to paint, format, and style the structure of your HTML page.
