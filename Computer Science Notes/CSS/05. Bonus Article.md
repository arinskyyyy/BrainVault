## **Pseudo-elements**

When we want to style our website, we usually create the elements in a **.html** file first and then style the selected elements in a **.css** file. But what if we could style certain parts of an element without adding more HTML?

For example, adding an emoji as a decoration rather a part of the actual text:

<img src="https://www.codedex.io/images/css/bonus-article-pseudo-element-example.png" />

We can do that with CSS pseudo-elements!

**Pseudo-elements** allow us to style parts of an element in fun and interesting ways, all without having to render new HTML. The "pseudo" part just means that we are giving CSS to a specific piece of the content.

Here is the syntax for pseudo-elements:

```css
selected-element::pseudo-element {
  /* CSS styles here */
}
```

Pseudo-elements use two colons `::` followed by its name.

Some use cases for pseudo-elements include:

- Creating decorations to go with labels or text.
- Styling the first letter or first line of a paragraph.
- Making custom tooltips that appear as you hover your mouse over something.
- And much more!

For example, if we wanted to include an emoji before or after some paragraph text, but we didn't want to include it in our HTML:

```css
p::after {
  content: "📂";
}
```

### **::first-letter & ::first-line**

The `::first-letter` and `::first-line` pseudo-elements select the first letter and first line of a piece of text, respectively. For example, consider the following HTML:

```html
<p>Welcome to the world of computing! We are so happy to have you here!</p>
```

If we styled this with CSS using the `::first-letter` and `::first-line` pseudo-elements, it would be something like the following:

```css
p {
  width: 25%;
}

p::first-letter {
  font-size: 36px;
  font-family: 'Gill Sans', sans-serif;
  text-decoration: underline;
  text-decoration-color: black;
  color: #e01a18;
}

p::first-line {
  font-size: 1.5em;
}
```

The rendered output would look like this:

<img src="https://www.codedex.io/images/css/bonus-article-pseudo-element-first-letter-line.gif" />

The paragraph itself is a quarter of the width of the page. As the page is resized, the characters wrapped to the bottom line are reduced to the default font size while the first line is 1.5 times the font size via its pseudo-element.

Additionally, we styled the first letter “W” of the paragraph content with a pseudo-element, setting its `font-size` to `36px`, giving it a `font-family`, a `text-decoration` of a black underline, and have it set to a hex code for a fancy red color.

### **::before & ::after**

The `::before` and `::after` pseudo-elements are arguably the most commonly used ones.

These pseudo-elements let us create, select, and style new content before and after the HTML-rendered content! This can be done in part with the special `content` property:

```css
selected-element::before {
  content: /* Content here */
  /* Other styles here */
}

selected-element::after {
  content: /* Content here */
  /* Other styles here */
}
```

Consider the following HTML:

```html
<p><a href="https://developer.mozilla.org/en-US/docs/Web/CSS/Pseudo-elements" target="_blank">Pseudo-elements</a> are keywords that we add to a selector that lets us style a specific part of the selected element(s).</p>
```

Here are two interesting ways to apply pseudo-elements to this HTML.

###  **Add a Tooltip**

You can use pseudo-elements along with the [`:hover` pseudo-class](https://www.codedex.io/css/bonus/pseudo-classes-in-css#inputs) to create a tooltip for an element! Assuming the `body` has a `relative` position and a `::before` pseudo-element has an `absolute` position:

```css
body {
  position: relative;
}

a:hover::before {
  content: "Matches when the user interacts with an element with a pointing device but does not necessarily activate it. It is generally triggered when the user hovers over an element with the cursor (mouse pointer).";
  position: absolute;
  top: 35px;
  bottom: 0;
  width: 15%;
  height: max-content;
  border: 2px solid;
  background-color: lightyellow;
  color: black;
  padding: 2px;
}
```

### **Add an Icon**

The first way is adding decorative icons at the ends of hyperlinked content that might sit within a paragraph of text:

```css
a::after {
  background-image: url(https://static-00.iconduck.com/assets.00/external-link-icon-256x256-m5z316qp.png);
  background-size: 10px 10px;
  display: inline-block;
  width: 10px;
  height: 10px;
  content:"";
  margin-left: 2.5px;
}
```

For the special `content` property, we use an empty set of quote marks. Instead, we're using the `url()`function to bring in an image on an external link icon.

### **Ground Rules**

We cannot use the same pseudo-element more than once on a selected element. For example, the following styles would be invalid:

```css
selected-element::before {
  content: "These";
  font-size: 1.5em;
}

selected-element::before {
  content: "Will not work";
  background-color: green;
}

selected-element::first-letter {
  font-family: "Helvetica";
}

selected-element::first-letter {
  background-color: purple;
}
```

Pseudo-elements must also be the last part of the selector. For example, the following would be invalid CSS:

```css
p::after.class {
  content: " This won't work either!";
  text-decoration: underline;
}
```

But this would be valid because `::after` comes after everything else in our style selector:

```css
p.class::after {
  content: " This will work!";
  text-decoration: underline;
}
```
