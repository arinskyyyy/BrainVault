### What is Html?

**HTML**, or **H**yper**T**ext **M**arkup **L**anguage. It was created by a developer named [Tim Berners-Lee](https://en.wikipedia.org/wiki/Tim_Berners-Lee) in 1991.

Html is just one piece used to build a web page. Most web pages use:

- HTML: Create the website skeleton.
