
## **links and Images**

### links :

**links** are integral to the idea of the internet. They are how users connect to other sites and navigate the web.

So how do we add a link to our web page?

We can use the `<a>` anchor element! This link tag allows us to add a [hyperlink](https://en.wikipedia.org/wiki/Hyperlink) to a piece of text.

```html
<a href="https://archive.org/web">Internet Archive</a>
```

Here, the `<a>` anchor element is similar to other tags, where the text inside is what is displayed. But what about the `href` inside the opening tag?

An `href`, or hyperlink reference, is a reference or pointer to another website that is linked in our HTML. When the text “Internet Archive” is clicked, the HTML will redirect to that site, which in our case, is [](https://archive.org/web)[https://archive.org/web](https://archive.org/web).

> [!NOTE]
> This can also be used to point to an email or phone number using a `mailto:`, `tel:` or  `sms:` parameter:
>
>- `<a href="mailto:frankie@gmail.com">📧</a>`
>- `<a href="tel:212-555-2368">🤙</a>`
>- `<a href="sms:320-250-HTML">💬</a>`

### Images :

What if we want to add images? We can use a similar format, but with the `<img>` image element.

```html
<p>Here's a cute pic:</p>
<img src="https://i.redd.it/5unn16axx1v81.jpg" width="250" height="250" alt="A preview of the image">
```

The `<img>` image element is another self-closing tag, so it doesn't have a closing tag.

- `src` attribute, which stands for “source”, specifies the file path of the image.
- `"https://i.redd.it/5unn16axx1v81.jpg"` is the image path.
- `alt` attribute, provides "alternative text" for an image, can be used when the image will not load, or shows an error.
- `width and height` attributes sets the width and height of the image.

> [!NOTE]
> *It is usually best practice to only set either `width` or `height` so the image maintains its original aspect ratio and doesn't stretch*

```html
<p align="center">
  <img src="https://www.codedex.io/images/editor/html-logo.png" width="100" height="200">
</p>
```

`align="center"` aligns the paragraph in the center.

For most images, the file path can be found by right-clicking on the image and selecting "Copy Image Address".
